Welcome to The HTML stylesheet

HTML is the standard markup language for creating Web pages.

Document Structure

tags

Tag Name Description
<html> html Root element of an HTML document
<head> head Contains meta-information about the document
<title> title Defines the title of the document
<body> body Contains the visible page content

attributes

Code Attribute Description
<html lang=""> lang Specifies the language of the document
<meta charset=""> charset Specifies the character encoding for the HTML document
<meta name="" content=""> name Specifies the name of the metadata
<meta name="" content=""> content Specifies the value of the metadata
<link rel="" href=""> rel Specifies the relationship between the current document and the linked document
<link rel="" href=""> href Specifies the location of the linked document

Structural Elements

tags

Tag Name Description
<header> header Defines a header for a document or a section
<nav> nav Defines a set of navigation links
<main> main Defines the main content of a document
<footer> footer Defines a footer for a document or a section
<section> section Defines a section in a document
<article> article Defines a self-contained composition in a document
<aside> aside Defines content aside from the content (like a sidebar)

attributes

Code Attribute Description
<header id=""> id Specifies a unique id for an element
<nav class=""> class Specifies one or more class names for an element
<main style=""> style Specifies an inline style for an element
<footer title=""> title Specifies extra information about an element
<section lang=""> lang Specifies the language of the element's content
<article dir=""> dir Specifies the text direction of the element's content
<aside tabindex=""> tabindex Specifies the tab order of an element

Text Elements

tags

Tag Name Description
<p> p Defines a paragraph
<h1> to <h6> h1 to h6 Defines HTML headings
<em> em Defines emphasized text
<strong> strong Defines important text

attributes

Code Attribute Description
<p style=""> style Specifies an inline style for an element
<h1 title=""> title Specifies extra information about an element
<em lang=""> lang Specifies the language of the element's content
<strong dir=""> dir Specifies the text direction of the element's content

Form Elements

tags

Tag Name Description
<form> form Defines an HTML form for user input
<input type=""> input Defines an input field for user input
<textarea> textarea Defines a multiline text area
<button> button Defines a clickable button
<select> select Defines a drop-down list
<option> option Defines an option in a drop-down list
<label> label Defines a caption for an input element
<fieldset> fieldset Groups related form elements in a form

attributes

Code Attribute Description
<form action="" method=""> action Specifies where to send the form-data when a form is submitted
<form method=""> method Specifies the HTTP method (GET, POST, PUT, DELETE, etc.) to use when sending form-data
<input type="" value=""> value Specifies the initial value for an input element
<textarea rows="" cols=""> rows Specifies the visible number of lines in a text area
<button type=""> type Specifies the type of button
<select name=""> name Specifies the name of an input element
<option value="" selected=""> value Specifies the value of an option element

Table Elements

tags

Tag Name Description
<table> table Defines an HTML table
<thead> table head Defines the head of an HTML table
<tbody> table body Groups the body content in an HTML table
<tr> table row Defines a row in an HTML table
<td> table data Defines a cell in an HTML table
<th> table head Defines a header cell in an HTML table

attributes

Code Attribute Description
<table border=""> border Specifies the width of the border around the table
<tr rowspan=""> rowspan Specifies the number of rows to merge cells vertically
<td colspan=""> colspan Specifies the number of columns to merge cells horizontally
<th scope=""> scope Specifies the header cell's scope within a table
<td headers=""> headers Specifies one or more header cells that contain the column headers for the current cell
