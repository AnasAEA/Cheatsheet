Welcome to The CSS Stylesheet

CSS is the language used to style HTML documents and create visually engaging web pages.

Basic Syntax

Selectors & Properties

Selector Description
element Selects all instances of a given element
.class Selects all elements with the specified class
#id Selects the element with the specified id
* Selects all elements

Properties

Property Description
color Sets the text color
background-color Sets the background color
font-size Sets the size of the text
margin Sets the margin around an element

Layout Techniques

Selectors & Properties

Property/Technique Description
display Defines how an element is displayed (block, inline, etc.)
position Specifies the positioning scheme (static, relative, absolute, fixed)
float Allows an element to float to the left or right
flex Enables flexible container layouts (Flexbox)

Attributes & Values

Property Value Description
display block Displays an element as a block-level element
display inline Displays an element as an inline element
position absolute Positions an element relative to its nearest positioned ancestor
position relative Positions an element relative to its normal position

Typography

Selectors & Properties

Property Description
font-family Specifies the typeface for text
font-size Defines the size of the text
font-weight Sets the thickness of the text (e.g., normal, bold)
line-height Sets the height of a line of text

Attributes & Values

Property Value Description
text-align center Centers the text horizontally
text-align left Aligns text to the left
text-align right Aligns text to the right
color #333 Sets the text color

CSS Specificity & Cascade

Selectors & Rules

Selector Specificity Description
element Low Targets all instances of an element
.class Medium Targets elements with the specified class
#id High Targets the element with the specified id
!important Overrides normal specificity Forces a style regardless of other specificity

Attributes & Rules

Property Example Description
margin margin: 10px; Sets the margin around an element
padding padding: 10px; Sets the padding inside an element
border border: 1px solid #000; Sets the border of an element
background-color background-color: #fff; Sets the background color of an element

Advanced CSS

Selectors & Techniques

Feature Description
@media Applies styles based on device characteristics
@keyframes Defines animations
transform Applies 2D or 3D transformations
transition Enables smooth property changes

Attributes & Techniques

Property Example Description
animation animation: slide 3s ease; Defines an animation
box-shadow box-shadow: 2px 2px 5px rgba(0,0,0,0.3); Adds shadow effects around an element
z-index z-index: 10; Sets the stacking order of elements
opacity opacity: 0.8; Sets the transparency level of an element

Display Types

Values and Descriptions

Display Value Description
block Element starts on a new line and occupies the full available width.
inline Element does not force a line break and only takes up as much width as needed.
inline-block Behaves like an inline element but accepts width and height properties.
flex Designates the element as a flex container, enabling flexible layouts.
grid Makes the element a grid container for two-dimensional layouts.
none Element is not displayed at all (removed from the layout).
table Renders the element as a block-level table.
table-row Displays the element as a table row, similar to the <tr> tag.
table-cell Displays the element as a table cell, similar to the <td> tag.
list-item Renders the element as a list item, complete with markers.
